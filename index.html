HISTORY OF JAVASCRIPT

ECMA Script 1 (1997)

  1. First standardized version of JavaScript, published by Ecma International.
  2. Defined language syntax, semantics, and built-in objects.
  3. Key Features:
    • Variables, data types, and operators.
    • Control structures(if/else, loops, switch).
    • Functions.
    • Objects(including Array, Dates, Math).

ECMA Scrpt 2 (1998)

  1. Minor updates, clarifications, and bug fixes.
  2. Improved compatability with ISO/IEC 16262.

ECMA Script 3 (1999)

  1. Significant Updates:
    • Regular expressions.
    • Try-catch blocks.
    • Switch statement.
    • Do-while loops.
    • Enhanced Object literals.

ECMA Script 4(Abandoned)

  1. Proposed in 2005, but abandoned due to disagreement among stakeholders.
  2. Some Features were later incorporated into ES6.

ECMA Script 5(2009)

  1. Major updates:
    • Strict Mode.
    • JSON support.
    • Function binding.
    • Array Methods (forEach, map, filter).
    • Enhanced object literals.

ECMA Script 5.1 (2011)

  1. Minor updates, clarifications, and bug fixes.
  2. Improved compatability with ISO/IEC 16262.

ECMA Script 6 (2015)

  1. Revolutionary updates:
    • Classes and inheritance.
    • Promises.
    • Generators.
    • Async/await.
    • Modules (import/export).
    • Let and const keywords.
    • Template literals.
    • Arrow functions.

Important Milestones

  1. 1995: Brendan Eich creates JavaScript.
  2. 2005: AJAX (Asynchronous JavaScript and XML) gains popularity.
  3. 2009: Node.js launched.
  4. 2010: HTML5 specification published.

Key Players

  1. Brendan Eich(Creator)
  2. Marc Andressen(Netscape co-founder)

Evolution of JavaScript

  1. Client-side scripting(1995)
  2. Server-side scripting(2009, Node.js)
  3. Mobile and desktop applications(2010s)
  4. IoT and embedded systems(2010s)
